# Immersive Birthday Experience 💕

A soft, romantic, and emotional birthday website with Korean drama vibes. Built with love.

## Features

- 🌙 Nighttime aesthetic with soft pink and warm brown tones
- ✨ Smooth animations with Framer Motion and GSAP
- 🎮 Interactive 3D chess experience
- 🎵 Ambient water sounds (spa-like atmosphere)
- 💫 Particle effects that respond to touch
- 📱 Fully responsive (mobile, tablet, desktop)
- 🎨 Floating words that drift gently
- 💝 15 unique screens with emotional storytelling

## Installation

```bash
npm install
```

## Run Development Server

```bash
npm run dev
```

Visit `http://localhost:3000`

## Build for Production

```bash
npm run build
```

## Tech Stack

- React 18
- Vite
- TailwindCSS
- Framer Motion
- GSAP
- Three.js / React Three Fiber
- Howler.js

## Experience Flow

1. Opening screen with video background
2. Personal reflections
3. Interactive choices
4. 3D Chess interaction with hidden messages
5. Color and emotion exploration
6. Dreams and aspirations
7. Connection and belonging
8. Final birthday message

## Notes

- Audio plays automatically (ambient water sounds)
- All interactions are smooth and gentle
- Hold/tap gestures reveal hidden messages
- Chess piece moves automatically with poetic responses
- Scene breathing effect (±1% scale) on all screens
